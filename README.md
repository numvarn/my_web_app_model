## เว็บแอปพลิเคชันแรก ที่พัฒนาด้วย Django Web Framework

ความรู้ที่ได้จากเรียนรู้

1. การสร้าง project ด้วย Django

```shell
> django-admin startproject <projectname>
```

2. การสร้าง python virtual environment ใน project

```shell
> python -m venv env
```
